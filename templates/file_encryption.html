Informasi Algoritma RSA
RSA Encryption

Algoritma asimetris dengan key pair:

  • Public Key: Untuk enkripsi
  • Private Key: Untuk dekripsi
Batasan Ukuran: RSA 512-bit hanya bisa mengenkripsi data maksimal 53 bytes. Cocok untuk kunci atau teks pendek.
Simpan kunci private dengan aman! Tanpa kunci private, file tidak bisa didekripsi.
Spesifikasi Teknis
  • Key size: 512 bit
  • Max Data Size: 53 bytes
  • Jenis: Asymmetric encryption
  • Kegunaan: Cocok untuk kunci, password, teks pendek
Tips Penggunaan
Untuk Teks Pendek
RSA cocok untuk password, kunci, atau teks pendek (max 53 karakter)
File Besar
Untuk file besar, gunakan enkripsi hybrid (AES + RSA)
Format Didukung
TXT, PDF, DOCX - untuk konten kecil
